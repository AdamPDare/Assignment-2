Creating reactive Columns

To create two reactive equal columns, on this webpage, I needed to set the width of each column to 50% and for both the columns to cover the whole webpage width. TO enable them to be reactive I used the @mediascreen feature which allowed me to dictate whether the webpage will display both columns for people viewing the website on a PC or just one column for people using a mobile phone. So the column positioning is affected by the window size of the webpage allowing the website to be more compatible with every device.

Creating the gallery

To create a responsive gallery, I used multiple media queries in which re-arranged the images and the navigation bar to facilitate devices with a small sized screen. This allows the website to be optimised for more devices and in turn appeal to a wider audience. Furthermore, I added a description to each image to say what they were as well as change the opactiy of three of the images when the mouse cursor is hovering over them. In the code extract below, it shows some of the CSS I used to create this gallery:

Navigation Bar with Dropdown

To create a navigation bar, I needed to used an unordered and normal list, the hover function and a dropdown button. These features allowed me to create a navigation bar that allows you to navigate each webpage on my website so far. I also utilised the @mediascreen feature to allow the navigation bars size change according to the size of the screen enabling my webpage to be used on a mobile device.

w3schools
