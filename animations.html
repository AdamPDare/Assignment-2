Animations

CSS animations enable you to gradually change the style of a html element to another. This is done via the use of the @keyframe rule. This is shown in the code extract below:

<style> 
    #animationBox{
        width: 100px;
        height: 100px;
        background: red;
        position: relative;
        animation: myFirst 5s linear 2s infinite alternate;
    }
                            
    @keyframes myFirst{
        0% {background-color: red;left: 0px; top: 0px;}
        25% {background-color: yellow;left: 200px; top: 0px;}
        50% {background-color: blue;left: 200px; top: 200px;}
        75% {background-color: green;left: 0px; top: 200px;}
        100% {background-color: red;left: 0px; top: 0px;}
    }
</style>

Upwards Animation

In the animation below, there is a delay before it starts. The squares will move up the page whilst changing colour.

The code extract below shows you the CSS behind the animation

    .animationDown{
        width: 100px;
        height: 100px;
        background-color: red;
        position: relative;
        animation-name: downWards;
        animation-duration: 5s;
        animation-delay: 5s;
        animation-fill-mode: forwards;
    }
    @keyframes downWards{
        from{top: 0px; background-color: yellow;}
        to{top: 300px; background-color: blue;}
    }

    .animationDownTwo{
        width: 100px;
        height: 100px;
        background-color: red;
        position: relative;
        animation-name: down;
        animation-duration: 5s;
        animation-delay: 5s;
        animation-fill-mode: forwards;
    }
    @keyframes down{
        from{top: 0px; background-color: yellow;}
        to{top: 400px; background-color: blue;}
    }
