Inline CSS

Inline CSS implementation is where the CSS code is applied to only a single HTML element. This CSS code must be added in the same line as typing the HTML tag and you must use the style element to apply the "style" you want to that specific element. Because of this, it makes Inline CSS less flexible as if you want to change the design of the element you have to edit them individually. An example of inline CSS is shown in the code snippet below:

<h3>Code Extract</h3>
            <h2 style="color:red"></h2>

Internal CSS

Internal CSS implementation is where the CSS code is found within the header section of the HTML page and within the style element. This CSS code will only affect the design of the webpage they are implemented into; this method of implementing CSS allows you to use classes and IDs. This enables you to style and design multiple HTML elements, on the webpage, without repeating lines of code as frequently. Also, using this method enables you to change the design of all the elements that are tagged on the webpage. An example of Internal CSS is shown below and it is the class used to format the code snippets on my webpage:

<h3>Code Extract</h3>
            <head>
                <style>
                    code{
                        font-family: monospace;
                        color: chocolate;
                                    
                    } 
                </style>
            </head>

External CSS

External CSS implementation is where you use a seperate Cascading Style Sheet (CSS) file; this will allow you to style your website and implement your CSS rules accross numerous HTML files and webpages. This is allows you to make changes to the CSS and it will implement them across the every HTML file that is linked to the CSS file, saving you time and effort. I used External CSS to style my Header, Navigation bar and my columns so I can easily make changes to it and it is implemented across each webpage that I have created. In the code snippet below it shows how I linked the CSS file to enable me to do this:

<h3>Code Extract</h3>
                    <head>
                        <link href="layoutStyleSheet.css" rel"stylesheet">
                    </head>

Creating reactive Columns

To create two reactive equal columns, on this webpage, I needed to set the width of each column to 50% and for both the columns to cover the whole webpage width. TO enable them to be reactive I used the @mediascreen feature which allowed me to dictate whether the webpage will display both columns for people viewing the website on a PC or just one column for people using a mobile phone. So the column positioning is affected by the window size of the webpage allowing the website to be more compatible with every device.

<h3>Code Extract</h3>
<style>
    code{
        .leftcolumn{
        float:left;
        width: 50%;
            padding-left: 20px;
            background-color: var(--main-bg-color);
        }
    
        .rightcolumn{
            float: left;
            width: 50%;
            background-color: var(--main-bg-color);
            padding-left: 20px;
        }
    
                
    
        .card{
            background-color: white;
            padding: 20px;
            margin-top: 20px;
    
        }
    
        div.card:hover{
            border: 5px solid var(--txt-border-color);
            box-shadow: 0 4px 8px 0 rgba(0, 0, 0, 0.2), 0 6px 20px 0 rgba(0, 0, 0, 0.19);
        }
    
        .row::after{
            content: "";
            display: table;
            clear: both;
        }    
        @media screen and (max-width: 800px){
            .leftcolumn, .rightcolumn{
                width: 100%;
                padding: 0;
            }
    
        }
        
                
    }
<style>

Navigation Bar with Dropdown

To create a navigation bar, I needed to used an unordered and normal list, the hover function and a dropdown button. These features allowed me to create a navigation bar that allows you to navigate each webpage on my website so far. I also utilised the @mediascreen feature to allow the navigation bars size change according to the size of the screen enabling my webpage to be used on a mobile device.

<h3>Code Extract</h3>
<style>
        ul{
            list-style-type:none;
            margin: 0;
            padding: 0;
            overflow: hidden;
            background-color: var(--main-bg-color);
        }
    
        li{
            float: left;      
            border-right: 1px solid var(--txt-border-color);
        }
    
        li a, .dropbtn{
            display: inline-block;
            color: var(--txt-border-color);
            text-align: center;
            padding: 14px 16px;
            text-decoration: none;
            
        }
    
        li a:hover:not(.active){
            border-bottom: 3px solid var(--txt-border-color);
            border-top: 2px solid var(--txt-border-color);
    
        }
        li a:hover, .dropdown:hover .dropbtn{
            border-bottom: 3px solid var(--txt-border-color);
        }
    
        li.dropdown{
            display: inline-block;
        }
    
        .dropdown-content{
            display: none;
            position: absolute;
            background-color: var(--main-bg-color);
            min-width: 160px;
            box-shadow: 0px 8px 16px 0px rgba(0,0,0,0.2);
            z-index: 1;
        }
    
        .dropdown-content a{
            color:var(--txt-border-color);
            padding: 12px 16px;
            text-decoration: none;
            display: block;
            text-align: left;
        }
    
        .dropdown-content a:hover{
            border-bottom: var(--txt-border-color);
        }
        .dropdown:hover .dropdown-content{
            display: block;
        }
        @media screen and (max-width: 600px) {
            .Navigation a{
                float:none;
                display: block;
                width: 100%;
                text-align: left;
            }
        }
