Basic Transitions

What is a transition

CSS transitions are a method used to change a property value, of a HTML element over a period of time. There are multiple properties to transitions: delay, duration, timing and property. These allow you to customise the characteristics of the transition.

W3schools Transitions

The example below shows a basic transition where the box size increases when you hover over it; will return to its original size afterwards.

The code snippet below, is how I created the basic transition

<style> 
    
    #transitionBox{
        width:100px;
        height:100px;
        background:aqua;
        transition: width 2s, height 4s;
                                
    }
    #transitionBox:hover{
        width: 300px;
        height:300px;
    }
</style>

All of the boxes below have different speed curves according to the timing function property

In the code extract below it shows how to apply the time function to the specific html element

    #timetransition1{transition-timing-function: linear;}
    #timetransition2{transition-timing-function: ease;}
    #timetransition3{transition-timing-function: ease-in;}
    #timetransition4{transition-timing-function: ease-out;}
    #timetransition5{transition-timing-function: ease-in-out;}
