CSS Box Model

What is the Box Model?

The CSS Box Model is the method used to design and layout the HTML elements in a webpage, as all the HTML elements are considered as boxes. These boxes have four layers/ parts: the content, padding, border and the margin; the last three parts/layers will encapsulate the actual content. The CSS code created will dictate the size, position and the properties of these CSS boxes allowing you to create an organised website layout.

What is the Content?

The content of the CSS box is the text, image or video that appears to the viewer/ user of the website. For example, the diagram above. The size and color of the content box can be adjusted also.

What is the Padding?

The padding of the CSS box is the transparent area surrounding the content; is normally the same color as the content box. The size of this box can be adjusted by the programmer and it prevents the content box from clashing and overlapping the border box and vice versa.

What is the Border?

The border can either be transparent or visible to the user; surrounds the padding and the content boxes. This allows the HTML elements visibly be seperated on the webpage. Borders can also be used to decorate the webpage.

What is the Margin?

The margin is the transparent outer box of the CSS box model and is the area that surrounds the border. This part of the CSS box is used to seperate each HTML element on the webpage, from each other, to prevent any clashes or overlapping.

CSS Selectors

CSS selectors are used to select or find the HTML elements that you want to style or apply CSS rules to. These selectors also enable you to group the elements you want to style, reducing the amount of repeated code.

Element Selector

Element selectors enable you to select a specific html elements by the elements name. For example: look at the code below: 

h2{
    color:aqua;
    font-size:25px;
    text-align:center;
}

ID Selectors

ID Selectors will assign a unique ID attribute to an HTML element. This ID selector will be unique on that specific webpage; must only be used to select that specific element. To create a ID in CSS you use a #. This is shown in the code below: 

#paragraph1{
    text-align: left;
    font-size:15px;
    color:white;
}

Class Selectors

Class Selectors are used to group/ select HTML elements and apply a specific CSS rule or attribute. To create a Class selector in CSS you must use a full stop/ dot. The code below, is the code is used to create the cards on this website: 

.card{
    background-color: white;
    padding: 20px;
    margin-top: 20px;

}

Universal Selectors

Universal Selectors are used to select every HTML element upon the webpage. To do this you use an asterisk * in the CSS code. This can be shown in the code below: 

*{
    box-sizing: border-box;
    background-color:red;
}

Grouping Selectors

Grouping selectors are used to group every HTML element that have the same style definitions/ element name. For example: h1,h2 or p. These selectors are shown in the code snippet below: 

h1{
    color: var(--txt-border-color);
    text-align: center;
    font: bold 200% fantasy;
}

ul{
    list-style-type:none;
    margin: 0;
    padding: 0;
    overflow: hidden;
    background-color: var(--main-bg-color);
}
li{
    float: left;
    border-right: 1px solid var(--txt-border-color);
}
