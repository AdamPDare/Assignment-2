CSS Box Model

What is the Box Model?

The CSS Box Model is the method used to design and layout the HTML elements in a webpage, as all the HTML elements are considered as boxes. These boxes have four layers/ parts: the content, padding, border and the margin; the last three parts/layers will encapsulate the actual content. The CSS code created will dictate the size, position and the properties of these CSS boxes allowing you to create an organised website layout.

What is the Content?

The content of the CSS box is the text, image or video that appears to the viewer/ user of the website. For example, the diagram above. The size and color of the content box can be adjusted also.

What is the Padding?

The padding of the CSS box is the transparent area surrounding the content; is normally the same color as the content box. The size of this box can be adjusted by the programmer and it prevents the content box from clashing and overlapping the border box and vice versa.

What is the Border?

The border can either be transparent or visible to the user; surrounds the padding and the content boxes. This allows the HTML elements visibly be seperated on the webpage. Borders can also be used to decorate the webpage.

What is the Margin?

The margin is the transparent outer box of the CSS box model and is the area that surrounds the border. This part of the CSS box is used to seperate each HTML element on the webpage, from each other, to prevent any clashes or overlapping.
